 📊 App User Behavior Segmentation Using Unsupervised Machine Learning

This project analyzes app user behavior using Unsupervised Machine Learning techniques.  
The main objective is to segment users based on their engagement and activity patterns using K-Means Clustering.

The project helps identify:
- High Engagement Users
- Moderate Users
- Low Engagement / At-Risk Users
- Occasional Users

An interactive Streamlit dashboard is also included for visualization and analysis

- Analyze user behavior data
- Perform Exploratory Data Analysis (EDA)
- Apply K-Means Clustering
- Visualize clusters using PCA
- Identify engagement patterns
- Generate business insights


 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit
- Plotly

 🔍 Workflow

1. Data Collection  
2. Data Cleaning  
3. Exploratory Data Analysis (EDA)  
4. Feature Selection  
5. Data Scaling using StandardScaler  
6. K-Means Clustering  
7. Elbow Method  
8. PCA Visualization  
9. Cluster Profiling  
10. Streamlit Dashboard  


📊 User Segments

🔹 High Engagement Users
- High activity
- Longer sessions
- Low churn risk

🔹 Moderate Users
- Average engagement
- Consistent usage

🔹 Low Engagement / At-Risk Users
- Low activity
- Higher churn risk

🔹 Occasional Users
- Irregular usage
- Low feature interaction

 📈 Dashboard Features

- KPI Metrics

- Churn Risk Analysis
- PCA Visualization
- Correlation Heatmap
- User Filters
